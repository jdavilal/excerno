# Installation Guide

## Dependencies

Make sure you have all of these packages installed already:

* BiocManager
* BSgenome
* ggplot2
* MutationalPatterns
* stringr
* bedr
* NMF
* BSgenome.Hsapiens.UCSC.hg38
* vcfR
* R.utils
* tidyverse

## Installing the package

```
install.packages("devtools")
devtools::install_github("jdavilal/excerno", dependencies = TRUE)
```
